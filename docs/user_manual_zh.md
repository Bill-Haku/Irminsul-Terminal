# 世界树终端（Irminsul Terminal） Discord Bot使用手册手册

这是为所有使用此对话机器人的普通用户提供的用户手册。在这里你可以知道每个命令的含义
以及如何使用它们。

## 首次使用时

你首先需要做的是将你的 Discord 帐户与你的原神的UID绑定。这样，机器人就可以知道 
你在游戏中是谁。

像这样使用该命令：

```
/bind [UID]
```

> 在输入UID时，你不需要输入`[]`。这些方括号意味着其中的内容是根据你的个人情况而变的。
> 在这里只代表其内容的类别。下同。

UID是一个整数，介于100000000和999999999之间。Discord会自动检查其是否合法。

接下来，由于我们支持3种不同的语言，你需要设置你的语言，机器人会记住它并以你的语言回答你的请求。

所有支持的语言是。
- 简体中文
- 英语
- 日语

> 对于英语用户，你很幸运，你可以跳过这一步，因为默认语言是英语。

像这样设置你的语言：

```
/setlanguage [语言]
```

对于这里的"语言"部分，你只需要从 Discord 提供的**选项**中选择。

## 核心功能

核心功能建立在你指定的想要查询的角色上。使用下面的命令来告诉机器人你想查哪个角色的相关信息。

```
/character [角色名称]
```

> 这里可以识别所有3种支持的语言的角色名称。尽管我们建议你使用正确的全名和
> 正确的拼写，我们已经考虑了许多常见的缩写和易错的字（中文或日文）。此外，在输入英文名字时，你不应该输入空格。
> 下面是一些例子。
> 
> - ✅: Ayaka
> - ✅: ayaka
> - ✅: kamisatoayaka
> - ❌: kamisato ayaka (没有空格)
> - ✅: 神里绫华
> - ✅: 绫华
> - ✅: 凌华 (错字)
> - ✅: 神里凌华 (错字)
> - ✅: 綾華
> - ✅: 神里綾華
> - ✅: 神里綾香 (错字)
> - ✅: あやか (假名)
> 

之后，机器人会回复你这个角色可用的所有功能按钮。你只需要点击按钮，机器人就会把结果回复给你。

你所有的数据都是从[Enka.Network](https://enka.network)的API中获得的，所有的
数据都会在你第一次使用后储存在服务器上，这意味着你的角色的数据将不会被更新，除非你使用`/sync`命令。

## 更新并同步你的角色数据

如果你在游戏中把一个新的角色放入你的角色展示柜中，或者你调整了某个角色的一些圣遗物或武器，你需要手动
让机器人同步数据。使用下面的命令：

```
/sync
```

**注意。你不能过于频繁地进行同步。相邻的两次同步请求之间应该间隔2分钟以上。

## 查询与你的 Discord 帐户绑定的信息

你可以使用以下命令来查询与你的 Discord 帐户绑定的信息。

```
/lookup
```

目前，你可以在数据库中查询到绑定的UID。只要点一下按钮即可。
